OSM Tools

An OpenStreetMap Toolkit for Python

OSMTools is a small set of Python tools for retrieving and using OpenStreetMap (OSM) images. Its original application was to draw a bunch of bus icons moving around on the map, which has been somewhat generalized.

With OSMTools you can:

  • Grab appropriate rendered OSM tiles from an OSM server
  • Patch tiles together into a bigger map
  • Easily animate stuff on that map (requires pygame)
